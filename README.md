# PetAid — Pet First-Aid Management System

A web-based prototype for pet first-aid instructions, search strategies, quizzes, and verification workflows.

## How to Run the App

Follow these simple steps to run the application on your computer:

### 1. Install Dependencies
Open your terminal in the project directory (`swe30003`) and install the required Python libraries:
```bash
pip install -r requirements.txt
```

### 2. Start the Server
Run the Flask application entry point:
```bash
python app.py
```

### 3. Open in Browser
Open your browser and navigate to:
[http://127.0.0.1:5000](http://127.0.0.1:5000)
